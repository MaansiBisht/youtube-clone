Project: YouTube Clone with React.js

Description: Developed a YouTube clone project using React.js, integrating the YouTube API for fetching video data. Utilized Redux Toolkit for state management and Tailwind CSS for styling.

Features:
    Homepage: Implemented a homepage with search functionality to find videos based on user queries.
    Video Playback: Enabled users to play videos directly within the application, with seamless playback functionality.

Technologies Used: React.js, YouTube API, Redux Toolkit, Tailwind CSS.
